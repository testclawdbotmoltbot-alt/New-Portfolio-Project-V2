## ğŸ” Your Data

### Where is it stored?
- Browser's local storage
- Per-browser (not synced between devices)
- Survives browser restarts
- Safe from hackers (client-side)

### How to backup?
- Use Export button in Settings
- Download happens automatically
- Save .json file to cloud storage
- Upload anytime to restore

### What if I lose it?
- Restore from backup.json file
- Create new demo portfolio
- Manually re-enter smaller portfolios
